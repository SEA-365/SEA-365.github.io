Java-运算符总结

Java-运算符总结

算术运算符

操作符 描述
+ 加法
- 减法
* 乘法
/ 除法
% 取余
++ 自增
– 自减

关系运算符

操作符 描述
== 等于
!= 不等于
> 大于
< 小于
>= 大于等于
<= 小于等于

位运算符

操作符 描述
& 按位与，（5&6） ==> (0101 & 0110) ==> 4
| 按位或，（5|6） ==> (0101 | 0110) ==> 0111 ===> 7
^ 按位异或，（5^6） ==> (0101 ^ 0110) ==> 0011 ==> 3
~ 按位取反，（~5） ==> 0101 ==> 1010(负数的补码转换成原码，按位取反加一) ==> -6
<< 按位左移，5<<2 ==> 0101 >> 2 ==>010100 ==> 20
>> 按位右移，5>>2 ==> 0101 >> 2 ==> 1
>>> 按位右移补零，5>>>2 ==> 0101 >>> 2 ==>0001 ==> 1

逻辑运算符

操作符 描述
&& 逻辑与
|| 逻辑或
! 逻辑非

赋值运算符

操作符 描述
= 简单的赋值
+= 加和赋值，A += B ===> A = A + B
-= 减和赋值
*= 乘和赋值
/= 除和赋值
%= 取模和赋值，A%B ===> A = A % B
<<= 左移位赋值，A<<=2 ===> A = A<<2
>>= 右移位赋值
&= 按位与赋值
^= 按位异或赋值
| 按位或赋值

其他运算符

操作符 描述
? : 三元运算符
instanceof 用于操作对象实例，检查该对象是否是一个特定的类型（类类型或累接口）

运算符优先级

表中，自上而下优先级依次降低。

类别 操作符 运算顺序
后缀 ()、 []、 .(点操作符) 自左向右
一元 a++、a– 自左向右
一元 ++a、–a、+、-、~、! 自右向左
乘 除 取模 *、/、% 自左向右
加 减 +、- 自左向右
移位 >>、>>>、<< 自左向右
关系 >、>=、<、<= 自左向右
相等 ==、!= 自左向右
按位与 & 自左向右
按位异或 ^ 自左向右
按位或 | 自左向右
逻辑与 && 自左向右
逻辑或 || 自左向右
条件 ? : 自右向左
赋值 =、+=、-=、*=、/=、%=、>>=、<<=、&=、|=、^=、、 自右向左
逗号 , 自左向右
