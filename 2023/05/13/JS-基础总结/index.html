变量 解释 示例

四、运算符

  • 一类数学符号，可以根据两个值（或变量）产生结果。
  • JavaScript中常用的运算符如下：
运算符 解释 符号 示例
加 将两个数字相加，或拼接两个字符串。 + 6 + 9;"Hello " + "world!";
减、乘、除 这些运算符操作与基础算术一致。只是乘法写作星号，除法写作斜杠。 -, *, / 9 - 3;8 * 2; //乘法在 JS 中是一个星号9 / 3;
赋值运算符 为变量赋值（你之前已经见过这个符号了） = let myVariable = '李雷';
等于 测试两个值是否相等，并返回一个 true/false （布尔）值。 === let myVariable = 3;myVariable === 4; // false
不等于 和等于运算符相反，测试两个值是否不相等，并返回一个 true/false （布尔）值。 !== let myVariable = 3;myVariable !== 3; // false
取非 返回逻辑相反的值，比如当前值为真，则返回 false。 ! 原式为真，但经取非后值为 false： let myVariable = 3;!(myVariable === 3); // false
幂 取底数的指数次方，即指数所指定的底数相乘。它在 EcmaScript 2016 中首次引入。 ** 5 ** 5 (返回 3125，相当于 5 * 5 * 5 * 5 * 5 。)
  • 注意：
    • 不同类型数据之间的计算可能出现奇怪的结果，因此必须正确引用变量，才能得出预期结果。
    • 比如：”25” + “35”或者25 + “35”的结果并不是60，而是字符串”25”与”35”进行拼接而成的”2535”。

5.5.1 let、var和const
作用域 变量提升 重复定义
let 块级，仅在定义的函数内可被访问 使用 let 声明的变量不会被提升，只有在变量声明后才能使用它 如果在同一个作用域内使用 let 定义相同的变量名，则会导致错误。
var 全局，在全局均可访问 var 声明的变量会在代码执行前被预处理，并在整个作用域可见 使用 var 声明的变量可以被多次定义，这将覆盖先前的定义。
const 也是块级，与let的区别在于：const修饰的变量是一个常量，不可更改； 与let一致 与let一致

七、字符串函数

  • 常用的字符串函数如下：
函数 描述 示例（let browserType = 'mozilla'; ）
length 获取字符串的长度 browserType.length;
indexOf(‘子串’) 在字符串中查找子字符串并提取它,找不到该子字符串则返回-1； browserType.indexOf('zilla');
slice(start, end) 已知字符串中的子字符串开始的位置，以及想要结束的字符时，slice()可以用来提取它。注意：范围是[start, end)区间；若只指定start，则返回的子字符串是下标从start开始的所有剩余字符。 browserType.slice(0,3);

Replace() 字符串替换 browserType.replace('moz','van');

8.2 常用函数

  • 如下：
    • let sequence = [1, 1, 2, 3, 5, 8, 13];
    • let myData = 'Manchester,London,Liverpool,Birmingham,Leeds,Carlisle';
函数 描述 示例
length 获取数组长度 sequence.length; // should return 7
split(‘分隔符’) 以指定分隔符分隔字符串返回分割后的数组 let myArray = myData.split(‘,’); //得到的myArray是一个长度为6的数组;
join(‘待插入分隔符’) 以指定待插入分隔符连接数组中的元素形成字符串 let myString = myArray.join(‘,’);//得到以待插入分隔符分隔的字符串；

push() 添加一个或多个要添加到数组末尾的元素。 myArray.push('Cardiff'); myArray.push('Bradford', 'Brighton');
pop() 从数组中删除最后一个元素 let removedItem = myArray.pop();
unshift() 添加一个或多个要添加到数组开头的元素。 myArray.unshift('Edinburgh');
shift() 从数组中删除开头的第一个元素 let removedItem = myArray.shift();
