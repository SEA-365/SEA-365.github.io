Java-运算符相关的类型转换

  • 当使用+,-,*,/运算符进行运算时，涉及操作数的数据类型转换遵循以下规则：

    • 1.自动类型提升：当参与运算的操作数有不同的类型时，会进行自动类型提升，将低精度类型转换为高精度类型。
    • 2.强制类型转换：仅能够将高精度类型转换为低精度类型，且可能导致精度丢失或截断。

  • Java虚拟机将byte、short和char类型均视为32位的int类型来进行操作；

  • Java中，整数常量默认为int类型，小数常量默认为double类型；
