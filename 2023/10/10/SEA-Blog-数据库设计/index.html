SEA-Blog-数据库设计

SEA-Blog-数据库设计

一、实体类设计

  • 博客项目实体类：
    • user：用户
    • article：文章
    • comment：评论
    • tag：文章标签
    • category：文章分类
    • operation_log：操作日志
    • login_log：登录日志
  • 实体类暂时先这么多；

二、表结构设计

  • 目前能想到的就这么多，原本按照规范的开发流程，在设计阶段应该确定好所有字段，避免后面开发过程中进行修改，但一个人写毕竟会有疏漏，暂时先这样设计，后面视情况增加修改；

2.1 用户表

  • user：用户

    字段名称 数据类型 长度 主键 允许空值 描述
    id INT 11 是 NOT NULL 主键，自增
    username VARCHAR 255 否 NOT NULL 用户名
    password VARCHAR 255 否 NOT NULL 密码
    email VARCHAR 64 否 NULL 邮箱
    phone BIGINT 11 否 NOT NULL 手机号码
    gender CHAR 2 否 NULL 性别
    intro VARCHAR 255 否 NULL 个人简介
    avatar_url VARCHAR 255 否 NOT NULL 头像图片url
    create_time DATETIME 0 否 NULL 创建时间
    update_time DATETIME 0 否 NULL 更新时间
